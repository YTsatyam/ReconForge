# Exposure & Attack Surface Assessment Platform (Python)

Enterprise-style Python tool to map exposed services, fingerprint basic technologies,
prioritize risk, track changes over time, and generate professional HTML reports.

## Features
- Automated service discovery (Nmap wrapper)
- Basic HTTP tech fingerprinting
- Risk scoring & prioritization
- Change detection between scans
- Scan history (JSON)
- Professional HTML reporting

## Skills Demonstrated
- Python automation
- Attack surface management concepts
- Risk analysis logic
- Security reporting
- Change tracking
